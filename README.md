# Tudat Experiments

This repository contains experiments and scripts using **TudatPy**
for orbital mechanics and space dynamics simulations.

## Structure
- `firstTest.py` – initial Tudat sanity test
- `scripts/` – main simulation scripts
- `notebooks/` – exploratory notebooks
- `results/` – generated outputs (not tracked)

## Environment
Developed using Anaconda + TudatPy.
